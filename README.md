- Multiple platforms to search from
- User friendly UI
- Easy to make custom revisions 
- Cross platform

## Run Locally

Clone the project

```bash
  git clone https://github.com/sim-byte/doxxer
```

Go to the project directory

```bash
  cd primary12.py
```

Install dependencies

```bash
  pip install requests 
  pip install time
```
